# green-house

A fullstack project about smart garden  
We use ReactJS, NodeJS, ExpressJS and MySQL in this project

### Installation
```
  git clone https://github.com/theksbd/green-house.git
```
Then head into each folder (client and server), type:
```
  npm install
```

### Run Client
```
  npm run client
```

### Run Server
```
  npm run server
```
